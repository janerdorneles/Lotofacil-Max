# Conferência Automática Lotofácil

Conferidor visual e inteligente para jogos da Lotofácil.

## Recursos

* Últimos 15 concursos (online)
* Quantidade de jogos (apostas): livre
* Conferência automática
* Destaque para prêmios e valores de premiação
* Interface responsiva
* 100% offline

## Como usar

1. Selecione dentre os últimos 15 concursos (online), ou...
2. Selecione manualmente 15 dezenas sorteadas (Modo offline/HTML)
3. Lance 1 ou mais jogos (limite 15 dezenas)
4. Clique em 'Importar Jogos'
5. Clique em "Conferir Resultados"
